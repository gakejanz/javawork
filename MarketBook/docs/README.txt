This app was written on Windows and should be run on windows.

There are instructions on how to employ the Database and tomcat servers.

From there just go to localhost:8080/MarketBook

Given, that your tomcat configuration has its' Connection Port Set to 8080.

This Program uses a SQL Database and Tomcat server to browse a database of Farmers Markets.

One can search by Zip, or City and State, as well as set a radius for their search.

Simply enter your search in the text field and hit enter to see the results.

Note if both the ZIp and City fields are occupied to app with default to searching the Zip.

NOTE: THE .WAR FILE IS IN THE SCRIPTS FOLDER

The Diagram of the database can be found in the docs directory.
